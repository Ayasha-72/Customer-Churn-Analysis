# Customer Churn Analysis

This project analyzes telecom customer data to identify factors contributing to customer churn.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Month-to-month customers have higher churn
- High monthly charges increase churn
- Short-tenure customers are more likely to leave

## Outcome
Provided data-driven business strategies to reduce customer churn.
